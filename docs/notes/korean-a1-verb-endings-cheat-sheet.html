Korean A1 – Verb Endings Cheat Sheet (Printable)

0) The idea

Most beginner sentences are: [Verb stem] + ending.

Dictionary form ends with -다 (가다, 먹다, 하다). In speech, you usually use polite endings like -아요/-어요.

Goal: pick the right polite ending fast (present / past / “I’m doing”).

1) Make the stem

Remove -다 → that’s the stem.

DictionaryStem
가다 (to go)가-
먹다 (to eat)먹-
하다 (to do)하-
보다 (to see/watch)보-

2) Present polite: -아요 / -어요

Look at the last vowel of the stem.

ㅏ or ㅗ → -아요 others → -어요

StemPresent
가-가요
보-봐요 (보+아요)
먹-먹어요
마시- (drink)마셔요

Beginner shortcut: If unsure, -어요 is usually safe.

3) The “do” verb: 하다 → 해요

하다 → 해요 (special contraction)

DictionaryPresentMeaning
하다해요do
공부하다공부해요study
일하다일해요work

4) Past polite: -았어요 / -었어요 / -했어요

Same vowel rule as present (ㅏ/ㅗ → 았어요, others → 었어요).

ㅏ or ㅗ → -았어요 others → -었어요 하다 → 했어요

DictionaryPast
가다갔어요
보다봤어요
먹다먹었어요
마시다마셨어요
공부하다공부했어요

5) “I’m doing”: -고 있어요

Use for actions happening now / ongoing.

MeaningExample
I’m studying공부하고 있어요.
I’m eating먹고 있어요.
I’m going가고 있어요.

Pattern: stem + 고 있어요

6) Mini drills (30 seconds)

  1. 먹다 → (present) ______ → 먹어요
  2. 가다 → (past) ______ → 갔어요
  3. 하다 → (present) ______ → 해요
  4. 보다 → (present) ______ → 봐요
  5. 공부하다 → (ongoing) ______ → 공부하고 있어요

You now can build: present / past / ongoing sentences.

7) Common A1 contractions (recognize fast)

보 + 아요 → 봐요 오 + 아요 → 와요 주 + 어요 → 줘요 마시 + 어요 → 마셔요 하 + 여요 → 해요

Don’t memorize every rule now—just recognize these patterns when you see/hear them.
