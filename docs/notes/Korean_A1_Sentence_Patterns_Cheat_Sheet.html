Korean A1 – Sentence Patterns Cheat Sheet (Printable)

0) Korean sentence order (default)

Subject + (Object) + Verb → Verb comes at the end.

저는 밥을 먹어요. 저는 집에 있어요. 저는 한국어를 공부해요.

Use particles to show roles (은/는, 이/가, 을/를, 에/에서).

1) “I am …” (noun)

Use 이에요/예요 (present polite).

PatternExamples
N(받침 O)+이에요 학생이에요. / 한국 사람이에요.
N(받침 X)+예요 의사예요. / 회사원이에요(often pronounced “예요” fast).

Common: 저는 학생이에요. / 이거 뭐예요?

2) “It is …” / “This is …”

PatternExamples
이거/그거/저거 + N 이거 커피예요. / 저거 책이에요.
N + 이에요/예요 이거는 물이에요. / 그거는 차예요.

이거 = this, 그거 = that (near you), 저거 = that (far)

3) “I have / don’t have”

Use 있다/없다.

PatternExamples
N이/가 있어요 시간이 있어요. / 돈이 있어요.
N이/가 없어요 시간이 없어요. / 차가 없어요.

Also “exist at”: 집에 있어요 (I’m at home).

4) “I want to …”

Use -고 싶어요 after the verb stem.

PatternExamples
V-고 싶어요 먹고 싶어요. / 자고 싶어요. / 가고 싶어요.

Want a thing: N을/를 먹고 싶어요 (I want to eat N).

5) “Please do …”

Use -아/어 주세요 (polite request).

PatternExamples
V-아/어 주세요 기다려 주세요. / 말해 주세요. / 도와 주세요.
Common fixed 주세요 = please give (also “please”).

In cafés: 아메리카노 하나 주세요 (One Americano, please).

6) “Can you …?” / “Can I …?”

Very common A1 polite pattern: -아/어도 돼요? (Is it okay if…?)

PatternExamples
V-아/어도 돼요? 들어가도 돼요? (May I enter?)
여기 앉아도 돼요? (May I sit here?)
Answer 네, 돼요. / 네, 괜찮아요. / 안 돼요.

7) Negation (not / don’t)

PatternExamples
안 + V 안 가요. / 안 먹어요. / 안 자요.
못 + V (cannot) 못 가요. (can’t go) / 못 해요. (can’t do)

안 = choose not to. 못 = cannot (ability/situation).

8) Micro drills (fill the blanks)

  1. 저는 학생_____. → 이에요
  2. 시간이 _____. → 없어요
  3. 커피를 먹_____. → 고 싶어요
  4. 여기 앉아____? → 도 돼요
  5. 오늘 ____ 가요. → 안

If you can use these 8 patterns, you can survive most A1 conversations.
